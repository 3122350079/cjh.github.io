xml文件创建虚拟机，旨在减少研发人员的重复性工作，提高效率。

痛点：
需要经常创建虚拟机，理论上需要注意的参数只有cpu，内存，磁盘，网卡。
但是操作过程中，需要注意磁盘类型，引导选项，网卡类型，显示协议，网桥选择等众多参数
而且virt-manager这个软件做的确实不怎么样，设置项少，修改不了默认配置。
每次创建虚拟机都需要配置很多不应该关注但必须要配置的选项。

目前做的比较简单，利用xml模板文件，使用virsh 命令创建虚拟机
创建一个基本可用的虚拟机出来，满足大部分需求，如果又特殊需求，可以创建完成后自己添加进去。


网卡确定两块：eth0 管理网   eth1 flat网络
磁盘确定两块：sda 系统盘   sdb 数据盘
如果网卡，磁盘数量需要加的，可以进行二次开发。

网卡类型都默认为virtio
磁盘类型arm的默认为scsi x86的默认为virtio

xml模板所需要参数 ： 
参数                名称                说明
虚拟机名称          NAMEMODE
内存大小            MEMORYSIZE          单位kb，容量和实际大小默认一致
cpu数量             CPUNUM 
系统盘qcow2路径     QCOW2PATH           本地的绝对路径
数据盘qcow2路径     DISKQCOW2PATH       本地的绝对路径
eth0 Mac地址        ETH0MAC             
eth0 绑定网桥       ETH0BRIDGE   
eth1 Mac地址        ETH1MAC             
eth1 绑定网桥       ETH1BRIDGE   



用户输入的参数
虚拟机名称
内存大小（Kb）
cpu数量
虚拟机盘路径（目录）
网卡绑定的网桥
